# Roadmap

## v1 - Basic LangGraph Server

**Goal:** Deliver a functional local server for running simple LangGraph chatbots
**Target:** Initial setup and basic functionality

### Features

**Chatbot Core** - IN PROGRESS

- User message input handling
- Conversation history maintenance
- Placeholder response generation

**LangGraph Server Integration** - IN PROGRESS

- Server setup and configuration
- API endpoints for graph execution
- Connection to LangGraph Studio

**Testing Framework** - IN PROGRESS

- Unit tests for agent logic
- Integration tests for server functionality
- CI/CD pipeline setup
