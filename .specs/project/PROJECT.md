# team-dev

**Vision:** A project to code projects and automate tasks using LangGraph.
**For:** Developers and teams
**Solves:** Automating coding workflows and task management

## Goals

- Provide a local server for running LangGraph-based applications
- Enable building conversational agents for coding assistance
- Support automation of development tasks

## Tech Stack

**Core:**

- Framework: LangGraph.js
- Language: TypeScript
- Runtime: Node.js

**Key dependencies:** LangGraph, Jest for testing

## Scope

**v1 includes:**

- Basic chatbot functionality with conversation history
- LangGraph server setup
- Integration with LangGraph Studio for debugging

**Explicitly out of scope:**

- Advanced AI integrations beyond basic chatbot
- Production deployment configurations
- Multi-user support

## Constraints

- Timeline: Not specified
- Technical: Based on LangGraph.js template
- Resources: Local development environment
